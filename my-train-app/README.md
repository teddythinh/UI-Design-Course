## Techstack

- React Native
- Expo
- React Navigation

## Installation

```bash
npm install
```

Note: If you use npm, install the dependencies with `npm install`. If you use Yarn, install the dependencies with `yarn`.

## Usage

```bash
npm start
```

This will start a development server for you.
